# Portfolio

My personal site — for anyone trying to figure out what I actually do, whether that's a recruiter, a hackathon teammate, or someone who found one of my Substack posts and wants the fuller picture. It covers ten years as an Amazon PM (with the shipped-impact numbers), what I've been building since with AI tools, my background, and a way to get in touch.

## How to use

Live: [neelanjanabasu.lovable.app](https://neelanjanabasu.lovable.app)

To run locally:
```bash
git clone https://github.com/neelanjanabasu13/neelanjanabasu.git
cd neelanjanabasu
npm i
npm run dev
```

## How it's built

Vite, React, TypeScript, shadcn-ui, and Tailwind CSS. Built on Lovable.
